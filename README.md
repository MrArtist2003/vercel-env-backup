# Vercel Env Backup Script for Linux (Kali/Ubuntu)

Quick bash script to pull and backup Vercel env vars on Linux. 

## Setup
1. Install Vercel CLI: 
npm i -g vercel@latest
2. Link project: vercel link --yes
3. Run: ./vercel-backup.sh in your project dir.

**Tested on Vercel CLI v33.6.0**. For dev backups—use fake secrets!
